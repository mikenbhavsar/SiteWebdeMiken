MATIERES ENSEIGNEES AU 2ND SEMESTRE :

Programmation impérative L’objectif de ce cours est d’apprendre le langage de programmation C à un niveau bas avec les outils de compilation, de debug et quelques bases d’algorithmique.
Programmation déclarative et bases de données Dans ce cours, nous apprenons plusieurs langages informatiques comme SQL qui exploite des bases de données relationnelles ainsi que Datalog et les bases de données déductives.
Architecture des ordinateurs Le but de ce cours est d’apprendre les principes de fonctionnement matériels d’un ordinateur afin de concevoir des programmes et logiciels.
Programmation dans les moteurs de jeux Son objectif est d'apprendre à implémenter des comportements complexes dans les scènes de jeux en allant plus loin à l'aide de la programmation en GDscript sur le logiciel Godot Engine.
Développement web et bases de données L'objectif de ce cours est d'apprendre à réaliser des applications web simples nécessitant une interaction client-serveur associée à une base de données.
Outils informatiques collaboratifs L’objectif de ce cours est de travailler en collaborant de façon à bien être organisé en utilisant un gestionnaire de contrôle de version (Git).
Anglais L’anglais est une langue fondamentale dans le domaine de l’informatique. En effet, beaucoup de documents n'existent qu’en anglais.
EC libre Un cours libre à choisir dans un autre département de l'université.
